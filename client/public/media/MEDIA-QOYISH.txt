   Yangi bitiruvchi rasmlari (masalan Lazizbek, Javohir Xalilov) — hozircha

3) Optimizatsiya (WebP)
   Yangi yoki o‘zgartirilgan jpg dan keyin terminalda:

     npm run optimize:media

   Har bir jpg yonida .webp yaratiladi; sayt <picture> orqali WebP ni
   ustun qiladi. Keyin: npm run build
